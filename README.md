# wxOfficialAccount
微信公众号文章批量保存到数据库

#### 介绍
微信公众号文章批量保存到数据库

#### 软件架构
软件架构说明
1.  jdk8+mysql5.4+springboot+jpa
2.	windows10+maven



#### 安装教程

1.  clone代码到本地，导入到idea
2.  在Test测试类中，将登录成功后的公众号total_article_data里面的list集合部分赋值给datas
3.  执行测试类即可
